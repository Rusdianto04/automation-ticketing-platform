# 🚀 Automation Ticketing Platform

Fullstack automation platform for ticketing and incident management.

## 📌 Components

### 🤖 Formbricks Discord Bot
- Automated ticket classification
- Incident detection
- Timeline progress automation
- Root cause analysis
- Prisma ORM integration

### 🌐 Portal Website
- User ticket submission
- Admin dashboard
- Incident handling interface
- Real-time ticket monitoring

## 🛠 Tech Stack
- Node.js
- Prisma ORM
- PostgreSQL
- Next.js
- Docker
- Discord.js

## 🎯 Purpose
Designed as a portfolio automation project for DevOps / Network / IT Automation roles.
